# About
Abyss is a communications full-stack web app focused on minimalism and privacy.

# Installation
### Frontend
```
cd frontend
pnpm install
```

### Backend
```
cd backend
pnpm install
```


# Running
### Frontend
```
cd frontend
pnpm dev
```

### Backend
```
cd backend
pnpm dev
```

# Prisma
### Creating new migration
```
cd backend
npx prisma migrate dev
```
### Pushing current prisma schema directly
```
cd backend
npx prisma db push
```
